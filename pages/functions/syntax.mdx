# Function Syntax 

## Function Decleration
The function decleration syntax in zura lang is very simple. The fn keyword then followed by the name of the function and the parameters in parentheses. The function block is then enclosed in curly braces. You can return any value with the return keyword. 

```zura
// This code will take in 2 parameters and return the sum of the 2 parameters
fn add(a, b) {
    return a + b;
}
```

## Function Call
To call a function, you just need to type the name of the function and the parameters in parentheses. 

```zura
// This code will call the add function and print the result to the console
info add(1, 2);
```
