# Command line arguments

Once you have installed the Zura programming language you can start to mess around with it. 
One of the most important things to know is how to run a Zura file. To do this you can do the following:

```bash
zura <file>
```

If you want to just open up the REPL you can do the following:

```bash
zura
```

If you want to see the version of Zura you are running you can do the following:

```bash
zura --version
```

If you want to see the help menu you can do the following:

```bash
zura --help
```

If you want to check the license of the software you can do the following:

```bash
zura --license
```
